# CYBORG NEXUS v3.5

AI-powered GUI Tool for UI/UX Design, Code Generation, Bug Scanning & Image Generation.

## Features

- 🎨 **UI_DESIGN** - Generate UI/UX designs with AI
- 💻 **CODE** - Generate code from descriptions  
- 🔍 **BUG_SCAN** - Analyze screenshots for UI bugs (drag & drop supported)
- 🖼️ **IMAGE** - Generate images from text prompts

## Requirements

```bash
pip install -r requirements.txt
```

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python main.py
   ```

## Usage

1. Select AI provider (Tgpt: sky, pollinations, deepseek, groq, ollama)
2. Enter your prompt in the input field
3. Select mode (UI/CODE/BUG/IMAGE)
4. Press ⚡ to execute
5. Press 💾 to save output

## Keyboard Shortcuts

- 📥 Upload image
- 💾 Save output
- 📋 Copy to clipboard
- ⚡ Execute
- 🧹 Clear output
- 📤 Open output folder

## Output

Generated files are saved in: `generated_outputs/`
